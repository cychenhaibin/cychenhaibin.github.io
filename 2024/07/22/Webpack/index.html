二、安装webpack

    cnpm install webpack -g

    cnpm install webpack-cli -g

  2. 测试版本

    webpack -v

    webpack-cli -v

四、loader

  1. loader是什么

    webpack只能理解Javascript和JSON文件，这是webpack开箱可用的自带能力。loader让webpack能够去处理其他类型的文件，并将他们转换为有效模块，以供应用程序使用，以及被添加到依赖图中

  2. loader的配置

    写法：

    1、test属性，识别出哪些文件会被转换

    2、use属性，定义出在进行转换时，应该使用哪个loader

五、插件

html-webpack-plugin

  1. 下载

    cnpm install html-webpack-plugin -S

  2. 引入

    var HtmlWebpackPlugin = require('html-webpack-plugin');

  3. 配置

  4. 参数说明

    1. 指定模版页面

      template:'./src/index.html'

    2. 修改默认输出文件名

      filename:'xx.html'

    3. 允许插入到模板中的一些chunk

      chunks:['jquery','xx']

    4. 配置不允许注入的thunk

      excludeChunks:['xx']

  5. 多页面打包

    需要创建多个new HtmlWebpackPlugin
